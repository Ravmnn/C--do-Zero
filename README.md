# Sobre

Este é um livro que criei (e, no momento, estou criando) como um hobby. Aqui, irei ensinar a programar na linguagem C++, considerando que você, leitor, não sabe nada a respeito de programação. Na realidade, você vai sair daqui (caso realmente leia o livro) também sabendo programar em Lua, uma linguagem de programação brasileira, que será utilizada para melhor demonstrar e aplicar lógica de programação, que também irei falar sobre. Mas não se engane, este livro não é apenas para iniciantes, pois quando digo que irei falar a respeito de C++, estou também incluindo sua funcionalidade por baixo dos panos (compiladores), o que consequentemente inclui a funcionalidade de computadores no geral e o papel do sistema operacional nisso tudo. Além disso, vou abordar boas práticas e lhe orientar a escrever código legível e limpo, ambas habilidades essencias para um verdadeiro programador.

- O Capítulo 1 lhe dá uma pequena introdução a respeito de linguagens de programação e do mundo computacional no geral.
- O Capítulo 2 fala sobre a lógica de programação, que é algo que praticamente todas as linguagens de programação possuem em comum. Este capítulo não é realmente necessário, mas ter um pouco de fundamento a respeito da lógica de programação pode ajudar a aprender linguagens mais rapidamente.
- O Capítulo 3 ensina a linguagem Lua, com objetivo de fixar melhor a lógica de programação. Decidi usar Lua para esse propósito porque é uma linguagem mais simples se comparada com C++.
- No Capítulo 4, finalmente iremos falar sobre C++. Atualmente não tem nada escrito lá.


Para começar a leitura, do Capítulo 1, [clique aqui](./book/introdução.md)
